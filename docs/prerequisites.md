## Haven Protocol Web App
# Prerequisites

### MacOS
Use [homebrew](https://brew.sh/)

```bash
> brew install node
```

if you have mac developer tools then you should already have git, if not you may want to grab it from homebrew

```bash
> git --version
```
If you get no response then 
```bash
> brew install git
```
